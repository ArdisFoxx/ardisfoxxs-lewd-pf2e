## Installation

1. In Foundry setup, go to the **Modules** tab and click **Install Module**
2. Paste the manifest URL and click Install:
   ```
   https://github.com/ArdisFoxx/ardisfoxxs-lewd-pf2e/releases/latest/download/module.json
   ```
3. Launch your world and activate **ArdisFoxx's Lewd PF2e**
4. Import the Arousal System items from the compendium, and the AFLP Core Macros into your world. You can leave everything else in the compendiums (the more you import from compendiums in Foundry, the slower your world loads, so its recommended to leave things in compendiums that aren't in use)
