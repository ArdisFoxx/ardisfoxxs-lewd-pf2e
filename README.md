## Installation

1. In Foundry setup, go to the **Modules** tab and click **Install Module**.
2. Paste the manifest URL and click Install:
   ```
   https://github.com/ArdisFoxx/ardisfoxxs-lewd-pf2e/releases/latest/download/module.json
   ```
3. Launch your world (Pathfinder 2e or Daggerheart) and activate **ArdisFoxx's Lewd Roleplaying**.
4. Import the Arousal System items and the AFLR Core Macros into your world. Leave everything else in the compendiums until you need it (importing fewer documents keeps your world loading fast).
